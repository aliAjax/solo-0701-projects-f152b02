# Solo 0701 Frontend Projects

当前仓库按做题人目录管理前端项目：

- `dfwl`：董飞
- `hxwl`：何鑫
- `hxywl`：黄心禹
- `wxwl`：王旭
- `xzwl`：徐哲

本次生成的项目都位于 `frontend` 目录下，每个项目都是独立的 Vite 前端应用，数据使用浏览器 `localStorage` 完成最小闭环。

## 运行方式

进入任意项目目录后执行：

```bash
npm install
npm run dev
```

构建检查：

```bash
npm run build
```
